## Como Executar os Desafios

### Java

Para executar um desafio em Java:

```sh
cd Java
javac Desafio1.java
java Desafio1
````

### C#
Para executar um desafio em C#:

```sh
cd CSharp
dotnet run Desafio1.cs
````
